# Citadel--Algo-WIDS

This repository contains a Week 1 implementation of a market microstructure
simulator based on a limit order book.

## Week 1 Objectives
- Understand how orders become trades
- Implement a minimal limit order book
- Study bid-ask spread, liquidity, and depth
- Design a modular, extensible exchange architecture

## Contents
- `week1/orderbook.py` – core limit order book implementation
- `week1/architecture/` – system architecture diagram
- `week1/report/` – Week 1 report and documentation

## Notes
This project focuses on exchange mechanics and system design.
Trading strategies and reinforcement learning agents will be added
in later stages.
